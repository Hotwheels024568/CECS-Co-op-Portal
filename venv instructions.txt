To create
	python3 -m venv venv

In venv\Scripts\Activate.ps1, put the following at the end of the file (before the signature block):
	# Enforces python to see the src folder and run code as a module
	$env:PYTHONPATH = "$env:VIRTUAL_ENV\..\src;$env:PYTHONPATH"

put the following at the end of the bat file (before :END):
	rem Enforces python to see the src folder and run code as a module
	set PYTHONPATH=%VIRTUAL_ENV%\..\src;%PYTHONPATH%

To run open the "Researcher Collab. Patterns Project" folder cmd
	To start the venv in cmd
		venv\Scripts\activate.bat
	
	To start the venv in PowerShell
		venv\Scripts\Activate.ps1
	
	To run
		python -m rcp.(folder).(file)
		python -m rcp.pdfs_and_csv.mcommunity_api
		fastapi run src/rcp/backend/main.py

To end
	deactivate
